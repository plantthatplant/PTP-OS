# ADR-004 — Morning Intelligence as the default interface

**Status:** Accepted · Sprint 0

## Context
Greenhouse work is shaped by a daily rhythm: growers begin the day deciding where
to focus. Traditional software answers this with dashboards — many screens of data
the grower must interpret under time pressure. That conflicts with our principles
("AI-first workflow rather than dashboards", "keep interfaces calm and simple",
"AI should explain its reasoning").

We need a default entry point that delivers value immediately and demonstrates
real value every day, while remaining usable on future hands-free interfaces (Even
G2 glasses).

## Decision
**Morning Intelligence is the default interface of PTP OS.** Instead of opening to
dashboards, the grower opens to a concise, prioritized, explained daily briefing:
what matters today, why, and what to do — generated by the pipeline (Context →
Decision → Language).

- It leads with answers and reasoning; raw data and charts are secondary support.
- Its content is structured first and rendered second, so it works on screen today
  and via voice/glasses later (no layer assumes a screen).

## Consequences
- A clear product north star: the daily briefing is where value is proven.
- Forces the whole pipeline to produce explained, prioritized output — reinforcing
  [ADR-001](ADR-001-separate-reasoning-from-language-generation.md).
- Naturally fits future hands-free interfaces.
- Cost: more upfront reasoning work than simply displaying data; quality of the
  briefing becomes the quality bar for the product. Accepted.

## Alternatives considered
- **Dashboard-first.** Familiar and cheaper to build, but pushes interpretation
  onto the grower and contradicts the AI-first, calm-interface principles.
  Rejected.
- **Chat-only assistant.** Flexible, but passive — it waits to be asked rather than
  proactively surfacing what matters each morning. Rejected as the default
  (conversational follow-up can complement Morning Intelligence).
- **Alert/notification feed.** Good for urgent events, but noisy and fragmented; it
  is not a calm, reasoned daily picture. Rejected as the default; alerts can feed
  into Morning Intelligence.
